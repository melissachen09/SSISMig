## 🚀 Quick Start

- dbt Core 1.7.0+
- Snowflake account and credentials

### Setup
1. **Configure your profile**:
   ```bash
   cp profiles.yml.template ~/.dbt/profiles.yml
   # Edit with your Snowflake credentials
   ```

2. **Install and test**:
   ```bash
   dbt deps
   dbt debug
   ```

3. **Run the models**:
   ```bash
   dbt run --target dev
   dbt test
   ```
